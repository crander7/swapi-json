Welcome to my swapi app

The available endpoints are /characters, /character/{name or id}, or /planetresidents

Also note that the /characters endpoint is sortable by applying "?sort={val}" to the end. Valid sort options are name, mass, and height.
The /character/{name or id} endpoint will match first names, last names, and full names. It also works with "-" or "_" between first and last names.
This node app is serving data from a cache that is updated every 30 minutes. If you wish to bypass the cache just add "?source=api" to any endpoint or "&source=api" if a query is already present.
Otherwise the cache is used by default unless the cache is older than 1 hour.
For ease of testing:
/characters?sort=mass
/characters?sort=name&source=api
/character/19
/character/C-3PO
/character/darth_vader
/planetresidents?source=api
